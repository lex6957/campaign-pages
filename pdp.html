1.  A pass starts automatically when connected to the applicable mobile network at the destination. It ends when the data allowance is used up or the specified validity period expires.
2.  Additional data passes could be bought under the same eSIM.
3.  If you have multiple passes in the same eSIM covering the same destination, the activation of next pass will be based on the purchase date, with the earliest purchase being activated first.
4.  The next pass will be automativelly start when the current pass ends. You can disable the eSIM on your device to avoid auto-activation.
